# Diretrizes de Acessibilidade WCAG 2.1 (Nível AA)

## Perceptível
- Alternativas textuais para imagens (alt text).
- Legendas para vídeos.
- Contraste mínimo 4.5:1 (texto) e 3:1 (componentes gráficos).
- Redimensionamento de texto até 200% sem perda de conteúdo.

## Operável
- Toda funcionalidade disponível por teclado.
- Tempo ajustável para conteúdos com limite de tempo.
- Não usar conteúdo que pisque mais de 3 vezes/segundo.
- Títulos e labels descritivos.

## Compreensível
- Idioma da página identificado.
- Navegação consistente.
- Mensagens de erro claras e sugestões de correção.

## Robusto
- Código HTML válido e semântico.
- Compatibilidade com leitores de tela.
