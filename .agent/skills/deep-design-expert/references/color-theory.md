# Teoria das Cores para Design de Interfaces

## Círculo Cromático
- Cores primárias (RGB no digital), secundárias, terciárias.
- Harmonia: complementares, análogas, triádicas, tetrádicas, monocromáticas.

## Psicologia das Cores
- Vermelho: urgência, paixão, perigo.
- Azul: confiança, calma, profissionalismo.
- Verde: natureza, saúde, crescimento.
- Amarelo: otimismo, energia, atenção.
- Roxo: luxo, criatividade, espiritualidade.
- Laranja: entusiasmo, amigável, chamativo.
- Preto: elegância, poder, sofisticação.
- Branco: pureza, simplicidade, espaço.

## Contraste e Acessibilidade
- WCAG exige contraste mínimo de 4.5:1 para texto normal, 3:1 para texto grande (18pt ou 14pt negrito).
- Ferramentas: WebAIM Contrast Checker.
