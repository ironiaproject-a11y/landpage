# Princípios da Gestalt aplicados ao Design de Interfaces

- **Proximidade:** Elementos próximos são percebidos como relacionados.
- **Similaridade:** Elementos com mesma forma/cor são vistos como grupo.
- **Continuidade:** A tendência de seguir uma linha ou curva.
- **Fechamento:** A mente completa figuras incompletas.
- **Figura-fundo:** Diferenciação entre elemento principal e fundo.
- **Pregnância:** A tendência de perceber formas simples e estáveis.

Aplicação: botões de ação devem ser similares entre si (similaridade), campos de formulário agrupados (proximidade), cabeçalhos destacados (figura-fundo).
